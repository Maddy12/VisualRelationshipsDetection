# VisualRelationshipsDetection
Uses PyTorch and OpenImages to detect visual relationships between objects. 

## Requirements
Basic requirements for PyTorch are the following:
* torch==1.1.0
* torchfile==0.1.0
* torchnet==0.0.4
* torchvision==0.3.0
